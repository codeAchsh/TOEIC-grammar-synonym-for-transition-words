# TOEIC Grammar: 穏やかに学ぶ英語のつなぎ言葉

このプロジェクトは、TOEIC対策として「前置詞」「接続詞」「接続副詞」などの英語のつなぎ言葉を、静かで感覚的に学べるWebページです。

## 特徴

- よく出題される類義語をボタンで分類
- クリックすると例文と日本語訳が表示されるミニ辞典形式
- 穏やかな色合いと直感的なUI
- GitHub Pagesで公開可能

## 使用技術

- HTML / CSS / JavaScript
- GitHub Pages
- Google Analytics（任意）
